Getting Started:


1. Extract all contents to a folder.
2. Connect the ChipKit unit.
3. Install MCB32 toolchain & follow the instructions (https://github.com/is1200-example-projects/mcb32tools/releases/)
4. Inside the MCB32 environment, navigate to the folder with the extracted contents
5. Execute "make && make install TTYDEV=/dev/...." The mcb32tools github page has instructions to find the path of your ChipKit, the path may vary on different systems.
